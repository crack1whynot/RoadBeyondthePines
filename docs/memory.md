# Memory Layer

The Memory layer stores long-term, structured project knowledge without executing work or performing planning.

## Responsibilities

- Store memory entries.
- Update and delete entries.
- Search and list entries.
- Track history of modifications.
- Create and restore snapshots.
- Load memory from disk.

## Design Principles

- Dependency injection throughout.
- Provider-independent persistence.
- Structured, serializable data.
- No AI or planning logic.
