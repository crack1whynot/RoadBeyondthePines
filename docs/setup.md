# Setup Guide

## Prerequisites

- Python 3.10+
- Node.js 20+
- npm

## Backend

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

## Frontend

```bash
npm install
npm run dev
```

## Environment

Copy .env.example to .env and update the values for your environment.
